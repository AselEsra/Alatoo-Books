In order to send or receive information we can make a specific “call” or “request”. It can also be used to update or delete data. There are 4 API methods which are GET, POST, PUT, DELETE. Below I have shown how I have used these API methods for my own data;


GET: Gathers information. ("books/all" is to get all books information and "books/id" is to get a single book's data by typing an id which we want to receive.)
POST: Sends data to the API server by creating new data. ("books/add", by this post request we create a new data by providing all of the information to body data.)
PUT: Updates pieces of data. ("books/id", we can change a single data by providing an id to put request which we want to update)
DELETE: Deletes pieces of data ("books/id" we can delete an unwanted book that we have created before by typing the id of that book.)
